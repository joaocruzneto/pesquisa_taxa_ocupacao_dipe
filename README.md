# pesquisa_taxa_ocupacao_dipe
Automação em Google Apps Script para o processamento, padronização e análise de dados da Pesquisa de Mancha e Taxa de Ocupação da DIPE (Divisão de Pesquisas). Otimiza o fluxo de tratamento de dados de frota e passageiros, reduzindo o processamento manual e garantindo a integridade dos cálculos operacionais do Grande Recife.
